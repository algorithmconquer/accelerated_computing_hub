## 第一部分:半导体光刻技术基础 (上午 10:20-12:20)

### 1.1 什么是光刻技术?

**光刻(Lithography)**是半导体制造中最关键的工艺步骤,通过光学系统将掩模版上的电路图案转移到硅片表面的光刻胶上。

#### 基本工作流程

```
光源 → 照明系统 → 掩模版 → 投影透镜 → 硅片(涂有光刻胶)
         ↓
    光刻成像过程
         ↓
    显影 → 图案形成
```

#### 关键技术参数

**瑞利判据**(决定分辨率):
```
CD = k₁ × (λ / NA)
```

其中:
- **CD**: 最小特征尺寸(Critical Dimension)
- **λ**: 光源波长(当前最先进的EUV光刻λ=13.5nm)
- **NA**: 数值孔径(NA = n × sinθ)
- **k₁**: 工艺系数(物理极限为0.25)

**提高分辨率的三条路径**:
1. 缩短波长: g-line(436nm) → EUV(13.5nm)
2. 提高NA: 0.35 → 1.35(浸没式) → 0.55(High-NA EUV)
3. 降低k₁: 通过计算光刻技术逼近物理极限

### 1.2 光刻技术的发展历程

| 年代 | 技术节点 | 波长 | 曝光方式 | 最小特征尺寸 |
|------|----------|----------|----------|--------------|
| 1960s | 接触式 | 436 | 接触式 | 7000 |
| 1970s | 投影式 | 365 | 投影式 | 2000 |
| 1980s | DUV | 248 | 投影式 | 450 |
| 1990s | DUV | 193 | 投影式 | 150 |
| 2000s | 浸没式 | 193+水 | 浸没式 | 45 |
| 2010s | 多重图形 | 193 | 浸没式+MP | 10 |
| 2018+ | EUV | 13.5 | 反射式 | 7及以下 |

### 1.3 光刻面临的核心挑战

#### 挑战1: 光学衍射效应

由于光的波动性,当特征尺寸接近光波长时,会产生严重的衍射效应:

```
目标图案:    ████████        (掩模上的理想图形)
              ↓ 光学系统
晶圆图案:    ▓▓▓▓▓▓▓▓        (因衍射而变形的图形)
            ████████
```

#### 挑战2: 分辨率极限

根据瑞利判据,当k₁趋近物理极限0.25时:
- 193nm浸没式光刻(NA=1.35): CD_min ≈ 36nm
- EUV光刻(NA=0.33): CD_min ≈ 10nm
- High-NA EUV(NA=0.55): CD_min ≈ 8nm

#### 挑战3: 工艺窗口

**焦深(DOF)**: 成像质量可接受的硅片位置变化范围
```
DOF = k₂ × λ / NA²
```

提高NA会提高分辨率,但同时会减小焦深,形成矛盾。

### 1.4 分辨率增强技术(RET)

为克服衍射极限,发展了一系列分辨率增强技术:

#### 1.4.1 光学邻近校正(OPC)

**传统OPC方法**:
- 基于规则的局部修正
- 添加辅助图形(serif、hammerhead等)
- 调整边的位置

```
原始掩模:    ████████
              ↓ OPC修正
修正掩模:    ▇▇▇▇▇▇▇▇
            ████████
              ↓ 光刻
晶圆图案:    ████████  (接近目标)
```

**OPC的局限性**:
- 基于规则,优化空间有限
- 难以处理复杂图案
- 工艺窗口较小

#### 1.4.2 逆光刻技术(ILT) - 核心创新

**ILT的核心思想**: 将光刻过程建模为正向模型,通过反向求解优化掩模图案

```
传统OPC流程:
设计图案 → 规则修正 → 掩模 → 光刻 → 晶圆图案

ILT流程:
设计图案 ← 反向优化 ← 掩模 → 光刻 → 晶圆图案
         (像素级全局优化)
```

**ILT的核心优势**:
1. **像素级优化**: 每个像素独立调整,优化自由度极大
2. **全局最优**: 通过梯度下降等方法寻找全局最优解
3. **曲线化掩模**: 生成曲线型掩模,适应High-NA EUV
4. **更大的工艺窗口**: 同时优化多个工艺角

### 1.5 ILT的数学模型

#### 正向光刻模型

**成像模型**(Hopkins部分相干成像):
```
I(x,y) = Σᵢ |hᵢ(x,y) ⊗ M(x,y)|²
```

其中:
- I(x,y): 晶圆上的光强分布
- M(x,y): 掩模图案
- hᵢ(x,y): 光学系统的核函数(SOCS分解)
- ⊗: 卷积运算

**光刻胶模型**(Sigmoid函数):
```
Z(x,y) = sigmoid(I(x,y) - threshold) = 1/(1 + e^(-α(I(x,y)-t)))
```

#### ILT优化问题

**目标函数**:
```
min J(M) = ||Z(M) - Z_target||² + λ₁·R(M) + λ₂·C(M)
```

其中:
- **Z(M)**: 光刻仿真得到的晶圆图案
- **Z_target**: 目标晶圆图案
- **R(M)**: 正则化项(约束掩模复杂度)
- **C(M)**: 约束项(像素值范围[0,1])

#### 梯度下降优化

```
算法流程:
1. 初始化掩模 M₀(通常使用目标图案)
2. for iteration = 1 to N:
   a) 正向光刻仿真: I = LithoSim(Mₖ)
   b) 计算损失: L = ||sigmoid(I) - Z_target||²
   c) 反向传播计算梯度: ∇M = ∂L/∂M
   d) 更新掩模: Mₖ₊₁ = Mₖ - η·∇M
   e) 应用约束: Mₖ₊₁ = clip(Mₖ₊₁, 0, 1)
```

### 2.1 为什么需要GPU加速ILT?

#### ILT的计算瓶颈分析

假设优化一个2048×2048分辨率的掩模:

**单次迭代计算量**:
- Hopkins模型卷积: 假设使用24个核函数
- 每个核函数: 2次FFT(正向和反向)
- 总FFT次数: 24 × 2 = 48次
- 每次FFT: O(N² log N) ≈ 40M次浮点运算
- 单次迭代总计算量: ≈ 2 GFLOPS

**总迭代次数**: 典型需要1000-5000次迭代

**总计算量**: 2-10 TFLOPS

**CPU运行时间**: 
- 单核CPU(3GHz, 峰值12 GFLOPS): 数小时到数天
- 多核CPU(16核): 几小时

**GPU运行时间**:
- 单块RTX 3090(峰值35 TFLOPS): 数分钟

**加速比**: 40-100倍

### 2.3 CUDA编程模型

#### 线程层次结构

```
Grid(网格)
├── Block(线程块) 0
│   ├── Thread(线程) 0
│   ├── Thread 1
│   ├── Thread 2
│   └── ...
├── Block 1
│   ├── Thread 0
│   ├── Thread 1
│   └── ...
└── ...
```

**关键概念**:
- **Thread**: 最小执行单元
- **Block**: 一组可协作的线程(通过共享内存通信)
- **Grid**: 多个Block的集合
- **Warp**: 32个线程,SM的调度单位

#### 内存层次结构

```
速度从快到慢:

寄存器(Registers)
└── 每个线程私有, 最快(~1 cycle)
    共享内存(Shared Memory)
    └── 线程块内共享, 快(~25 cycles)
        L2缓存
        └── 所有SM共享(~100 cycles)
            全局内存(Global Memory/HBM)
            └── 容量大但慢(~500 cycles)
```

**RTX 3090内存性能**:
- 寄存器: 256KB/SM
- 共享内存: 100KB/SM
- L2缓存: 6MB
- 全局内存(HBM2): 24GB, 带宽936GB/s

### 2.4 CUDA核函数示例

#### 第一个CUDA程序:向量加法

```cpp
// vector_add.cu
#include <cuda_runtime.h>
#include <stdio.h>

// CUDA核函数 - 在GPU上执行
__global__ void vectorAdd(float *a, float *b, float *c, int n) {
    // 计算全局线程索引
    int index = blockIdx.x * blockDim.x + threadIdx.x;
    
    // 边界检查
    if (index < n) {
        c[index] = a[index] + b[index];
    }
}

int main() {
    int n = 1000000;
    size_t size = n * sizeof(float);
    
    // 分配主机内存
    float *h_a = (float*)malloc(size);
    float *h_b = (float*)malloc(size);
    float *h_c = (float*)malloc(size);
    
    // 初始化数据
    for (int i = 0; i < n; i++) {
        h_a[i] = i;
        h_b[i] = i * 2;
    }
    
    // 分配设备内存
    float *d_a, *d_b, *d_c;
    cudaMalloc(&d_a, size);
    cudaMalloc(&d_b, size);
    cudaMalloc(&d_c, size);
    
    // 数据传输: Host → Device
    cudaMemcpy(d_a, h_a, size, cudaMemcpyHostToDevice);
    cudaMemcpy(d_b, h_b, size, cudaMemcpyHostToDevice);
    
    // 启动核函数
    int blockSize = 256;
    int numBlocks = (n + blockSize - 1) / blockSize;
    vectorAdd<<<numBlocks, blockSize>>>(d_a, d_b, d_c, n);
    
    // 数据传输: Device → Host
    cudaMemcpy(h_c, d_c, size, cudaMemcpyDeviceToHost);
    
    // 验证结果
    for (int i = 0; i < 5; i++) {
        printf("%f + %f = %f\n", h_a[i], h_b[i], h_c[i]);
    }
    
    // 释放内存
    free(h_a); free(h_b); free(h_c);
    cudaFree(d_a); cudaFree(d_b); cudaFree(d_c);
    
    return 0;
}
```

import torch
import torch.fft as fft

def litho_simulation(mask, kernels):
    """可微分的光刻仿真"""
    M_freq = fft.fft2(mask)
    intensity = torch.zeros_like(mask)
    
    for kernel in kernels:
        H_freq = fft.fft2(kernel)
        result = fft.ifft2(M_freq * H_freq)
        intensity += torch.abs(result)**2
    
    return intensity

            torch.zeros_like(target_pattern, device=device)
        )
    
    def litho_simulation(self, mask):
        """简化的光刻仿真(使用高斯模糊近似衍射效应)"""
        # 创建简单的高斯核
        kernel_size = 31
        sigma = 5.0
        x = torch.arange(kernel_size, device=self.device) - kernel_size // 2
        kernel_1d = torch.exp(-x**2 / (2 * sigma**2))
        kernel_2d = kernel_1d.unsqueeze(0) * kernel_1d.unsqueeze(1)
        kernel_2d /= kernel_2d.sum()
        
        # FFT卷积
        mask_padded = torch.nn.functional.pad(
            mask.unsqueeze(0).unsqueeze(0),
            (kernel_size//2, kernel_size//2, kernel_size//2, kernel_size//2),
            mode='reflect'
        ).squeeze()
        
        mask_freq = fft.fft2(mask_padded)
        kernel_freq = fft.fft2(kernel_2d, s=mask_padded.shape)
        result = fft.ifft2(mask_freq * kernel_freq).abs()
        
        # 裁剪回原始大小
        return result[:mask.shape[0], :mask.shape[1]]
    
    def optimize(self, iterations=100, lr=0.5):
        """优化掩模"""
        optimizer = torch.optim.Adam([self.mask_param], lr=lr)
        losses = []
        
        for i in range(iterations):
            optimizer.zero_grad()
            
            # 通过sigmoid确保掩模值在[0,1]范围
            mask = torch.sigmoid(self.mask_param)
            
            # 光刻仿真
            wafer = self.litho_simulation(mask)
            
            # 计算损失(L2距离)
            loss = torch.mean((wafer - self.target)**2)
            
            # 反向传播
            loss.backward()
            optimizer.step()
            
            losses.append(loss.item())
            
            if i % 10 == 0:
                print(f"Iteration {i}: Loss = {loss.item():.6f}")
        
        return torch.sigmoid(self.mask_param), losses

def create_target_pattern(size=256):
    """创建目标图案(简单的十字形)"""
    pattern = torch.zeros(size, size)
    center = size // 2
    width = size // 10
    
    # 横线
    pattern[center-width//2:center+width//2, center-size//4:center+size//4] = 1.0
    # 竖线
    pattern[center-size//4:center+size//4, center-width//2:center+width//2] = 1.0
    
    return pattern

def main():
    print("=" * 60)
    print("第一个GPU加速逆光刻技术(ILT)演示程序")
    print("=" * 60)
    
    # 创建目标图案
    target = create_target_pattern(256)
    print(f"\n目标图案大小: {target.shape}")
    
    # 创建ILT优化器
    ilt = SimpleILT(target, device='cuda')
    
    # 执行优化
    print("\n开始优化...")
    optimized_mask, losses = ilt.optimize(iterations=100, lr=0.5)
    
    # 光刻仿真验证
    final_wafer = ilt.litho_simulation(optimized_mask)
    
    # 可视化结果
    fig, axes = plt.subplots(1, 4, figsize=(16, 4))
    
    axes[0].imshow(target.cpu().numpy(), cmap='gray')
    axes[0].set_title('目标图案\n(Target Pattern)')
    axes[0].axis('off')
    
    axes[1].imshow(optimized_mask.detach().cpu().numpy(), cmap='gray')
    axes[1].set_title('优化后的掩模\n(Optimized Mask)')
    axes[1].axis('off')
    
    axes[2].imshow(final_wafer.detach().cpu().numpy(), cmap='gray')
    axes[2].set_title('晶圆上的图案\n(Wafer Pattern)')
    axes[2].axis('off')
    
    axes[3].plot(losses)
    axes[3].set_title('优化过程\n(Optimization Loss)')
    axes[3].set_xlabel('Iteration')
    axes[3].set_ylabel('Loss')
    axes[3].grid(True)
    
    plt.tight_layout()
    plt.savefig('first_ilt_result.png', dpi=150, bbox_inches='tight')
    print("\n结果已保存到: first_ilt_result.png")
    
    # 打印统计信息
    print("\n" + "=" * 60)
    print("优化统计:")
    print(f"  初始损失: {losses[0]:.6f}")
    print(f"  最终损失: {losses[-1]:.6f}")
    print(f"  损失降低: {(1 - losses[-1]/losses[0])*100:.2f}%")
    print("=" * 60)

if __name__ == "__main__":
    main()
